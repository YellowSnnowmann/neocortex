## Neocortex & Alphahuman Integrations

This folder contains all **3rd party SDKs and integrations** for Neocortex / Alphahuman memory.

- **Python SDK** — `packages/sdk-python/`  
  Neocortex Python SDK for building AI apps with persistent memory on TinyHumans / Neocortex.  
  See `packages/sdk-python/README.md`.

- **LangGraph SDK** — `packages/sdk-lang-graph/`  
  LangGraph integration for the Alphahuman Memory API, exposing memory tools as LangGraph nodes / LLM tools.  
  See `packages/sdk-lang-graph/README.md`.

- **TypeScript SDK** — `packages/sdk-typescript/`  
  TypeScript / JavaScript client for the Alphahuman Memory API, suitable for Node.js and browser apps.  
  See `packages/sdk-typescript/README.md`.

New integrations should be added as **new subfolders** under `packages/`, each with its own `README.md` and release process.
